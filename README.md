# 1-19

這是一個純前端的背單字學習應用，使用 HTML、CSS、JavaScript 開發，且沒有套用任何前端框架。

## 功能

- 卡片式翻頁學習介面
- 單字正面顯示英文，背面顯示翻譯、詞性、例句、字根分析
- 獨立管理頁面，可新增、編輯、刪除單字
- 管理頁提供「自動填入」按鈕，可呼叫公開 API 自動填入翻譯與詞性、例句、字根分析等欄位
- 內容會儲存在瀏覽器 localStorage

## 使用方式

1. 開啟 `index.html`
2. 點選「管理頁」新增單字
3. 若要自動填寫資料，輸入英文單字後按「自動填入」
4. 儲存後前往「學習頁」點擊卡片翻頁

## 檔案

- `index.html`：頁面結構
- `styles.css`：介面樣式
- `script.js`：功能邏輯與資料管理
